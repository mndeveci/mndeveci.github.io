İşletim sisteminde varsayılan olarak atanmış program ile dosya açmak için aşağıdaki komutları kullanabilirsiniz. 

Mac OS için; 

<pre lang="java"> 
open filename.pdf 
</pre>

Microsoft Windows için; 

<pre lang="java"> 
start filename.pdf 
</pre>

Linux dağıtımları için; 

<pre lang="java"> 
xdg-open filename.pdf 
</pre>
